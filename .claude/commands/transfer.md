# Claude 命令：翻译

将指定的 Markdown 文件翻译为中文，并输出到 `output/` 目录。

## 用法

```
/transfer <文件名.md>
```

示例：
```
/transfer commit.md
```

## 此命令做了什么

1. 读取指定的 Markdown 文件
2. 将文件内容翻译为中文：
   - 保留原有的结构和层级
   - 保留所有 emoji 和代码块
   - 代码块内容和命令保持原样
   - 类型标识符（如 `feat`、`fix` 等）保留原文
3. 创建 `output/` 目录（如果不存在）
4. 将翻译后的内容写入 `output/<文件名>.md`

## 重要说明

- 只翻译 Markdown（.md）文件
- 如果 `output/` 目录已存在同名文件，将被覆盖
- 翻译完成后会显示输出文件路径
